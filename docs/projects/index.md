# Projects
My projects concern optimization.
Optimization problems from university are usually small, eagle-eyed readers may find the optimal solution just by looking, or, by inserting any real number $$ r \in [-2,2] $$.
Optimization problem from research are usually terribly big, hard to solve and are meant to push the boundaries of what can be solved.
The problems that I aim present have real-world size and always an application/problem-driven motivation.
The code that I write is always available on my github.
I use open source optimization software.
Problems are written in [JuMP](https://jump.dev/JuMP.jl/stable/) using various solvers.
Sometimes I compare JuMP to other modeling languages/solvers such as [Gurobi](https://www.gurobi.com/).
